# 🐱 План разработки лендинга для кошки

## 📋 Обзор проекта

**Цель:** Создать яркий, мемный и вайбовый лендинг-обзор на кошку с фото и видео контентом.

**Аудитория:** Семья (закрытый доступ, для развлечения)

**Настроение:** Весёлый, динамичный, с юмором и анимациями

---

### Основной стек

## 🎨 Структура и секции лендинга

### 1. Hero секция (Главная)
- **Контент:**
  - Большое фото кошки вFullScreen
  - Имя кошки + забавный слоган
  - Анимированные элементы (парящие лапки, сердечки)
  - Bounce/fade-in анимации при загрузке
  
- **Фичи:**
  - Parallax эффект на фоне
  - Gradient overlay для читаемости текста
  - Кнопка "Познакомиться ближе" (плавный скролл вниз)

### 2. "О звезде" (About)
- **Контент:**
  - Карточка с фото профиля кошки
  - Интересные факты (возраст, порода, любимая еда)
  - Смешные характеристики (сила мяуканья, уровень драмы)
  - Progress bars с анимацией для характеристик

- **Фичи:**
  - Карточка с hover-эффектами
  - Анимированные статы при появлении в viewport
  - Ротация фото при наведении

### 3. "Фотогалерея" (Gallery)
- **Контент:**
  - Адаптивная masonry/grid галерея фотографий
  - Разные размеры карточек для динамики
  - Подписи с юмором под каждым фото

- **Фичи:**
  - Lightbox для просмотра в полном размере
  - Hover-эффекты и zoom
  - Lazy loading изображений
  - Фильтры по категориям (сон, игры, драма, еда)

### 4. "Видео моменты" (Videos)
- **Контент:**
  - Сетка из видео карточек
  - Превью видео с play кнопкой
  - Забавные названия видосов

- **Фичи:**
  - Модальное окно для просмотра видео
  - Автопаузе при закрытии модалки
  - Плавные переходы

### 5. "Мемная хронология" (Timeline)
- **Контент:**
  - Вертикальная временная линия с ключевыми событиями
  - Смешные даты (первое мяу, первый зумис и т.д.)
  - Мини фото/видео к каждому событию

- **Фичи:**
  - Анимация появления элементов при скролле
  - Чередующееся расположение (слева-справа)
  - Sticky навигация по датам

### 6. "Достижения" (Achievements)
- **Контент:**
  - Бейджи/медали с иконками
  - Смешные награды (Мастер сна, Охотник за лазерами)
  - Animated counter для статистики

- **Фичи:**
  - Карточки с flip-анимацией
  - Конфетти эффект при клике на достижение
  - Hover эффекты с тенями

### 7. Footer
- **Контент:**
  - Фото подпись "Сделано с любовью"
  - Год создания
  - Опционально: fake социальные сети кошки

## 🎮 Интерактивные фишки и микровзаимодействия

### Scroll-based анимации
1. **Вращающаяся кошка при скролле** 
   - Иконка/изображение кошки в углу экрана, которая поворачивается в зависимости от scroll position
   - Rotate от 0 до 360° синхронно со скроллом

2. **Параллакс элементы**
   - Разные слои двигаются с разной скоростью
   - Фоновые лапки "плывут" медленнее, чем основной контент

3. **Progress bar в виде кота**
   - Котик "идёт" по верху страницы показывая прогресс чтения
   - Меняет позу в зависимости от прогресса (сидит → идёт → бежит)

### Активные элементы на странице
4. **Бегающие элементы**
   - Рандомно появляющаяся мышка/игрушка, которая пробегает по экрану
   - Лазерная указка, за которой можно "водить" курсором
   - Летающая бабочка в определенных секциях

5. **Плавающие частицы**
   - Падающие лапки, сердечки, рыбки в фоне
   - Интенсивность зависит от секции (в галерее больше)
   - Можно кликать на них (исчезают с анимацией)

6. **Интерактивный котик-компаньон**
   - Маленький спрайт кошки следует за курсором с задержкой
   - Реагирует на клики (прыжок, мяу)
   - Можно включить/выключить кнопкой

### Hover эффекты
7. **Следящие глаза**
   - У некоторых фото кошки глаза "следят" за курсором
   - Используя transform: rotate на элементах глаз

8. **Шевеление ушей**
   - При наведании на фото кошки уши немного двигаются
   - CSS анимация + hover trigger

9. **Tail wagging** (виляние хвостом)
   - У элементов-иконок кошки хвост двигается при hover

### Клик-эффекты
10. **Лапки на кликах**
    - При клике в любом месте страницы появляется след лапки
    - Fade out после появления
    - Разные размеры и ротации для разнообразия

11. **Мяуканье**
    - При клике на определенные элементы короткий звук мяу
    - Можно включить/выключить звук кнопкой

12. **Confetti burst**
    - При достижении определенных секций
    - При клике на achievements
    - Разноцветные лапки вместо стандартного конфетти

### Easter Eggs
13. **Секретные триггеры**
    - Тройной клик по лого → кот делает сальто
    - Konami code → секретная фотка/видео/мем
    - Долгий hover на определенном элементе → сюрприз

14. **Время суток**
    - Если зайти ночью — кот спит в углу
    - Днем — активный и игривый
    - Вечером — потягивается

### Технические фишки
15. **Кастомный курсор**
    - Обычный: лапка
    - При hover на кликабельное: лапка с когтями
    - При драге: закрытая лапка

16. **Shake эффект**
    - При долгом бездействии — кот "трясёт" экран (легкая вибрация)
    - Привлекает внимание к интерактивным элементам

17. **Typing indicator**
    - В секции About при загрузке "печатается" описание
    - Как будто кот сам рассказывает о себе

## 💡 Дополнительные креативные идеи

## 📝 Важные примечания

### Оптимизация контента
- Все изображения должны быть в формате WebP для оптимизации
- Использовать `next/image` для автоматической оптимизации

- Для длинных видео лучше использовать YouTube embed
- Lazy loading для всех медиа ниже fold

### Производительность
- Использовать dynamic imports для тяжелых компонентов
- Particles/конфетти должны быть оптимизированы (лимит элементов)
- Звуки предзагружать только после первого взаимодействия
- React.memo для компонентов, которые часто re-render'ятся

### Доступность (A11y)
- Добавить `alt` текст для всех изображений
- Keyboard navigation для галереи и модалок
- ARIA-labels для интерактивных элементов
- Focus states для всех кликабельных элементов
- Кнопка "Пропустить анимации" для пользователей с motion sensitivity

### Мобильные устройства
- **Кастомный курсор** — отключить на touch devices
- **Hover эффекты** — заменить на tap/click на мобильных
- **Звуки** — по умолчанию выключены (autoplay политика браузеров)
- **Particles** — уменьшить количество на слабых устройствах
- **Видео** — меньшее разрешение для мобильных

### SEO и метатеги
- Добавить Open Graph метатеги для красивых превью
- Twitter Card метатеги
- Favicon сделать в виде мордочки кошки (PNG + SVG)
- Добавить описание и title для каждой секции

### Безопасность
- Не коммитить `.env.local` в Git
- Если будут API ключи (для звуков/CDN) — использовать environment переменные
